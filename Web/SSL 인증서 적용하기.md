# SSL 인증서 적용하기

## 레퍼런스 참조하여 공부중
### SSL이란?
우선 `SSL`이란 서버와 사용자(브라우저) 간의 통신을 할 경우 정보를 암호화하고 도중에 해킹을 통해 정보가 유출이 된다고 하더라도 정보의 내용을 보호할 수 있는 보안 인증 솔루션 기술이라고 생각하면 됩니다.

### 인증서 종류
